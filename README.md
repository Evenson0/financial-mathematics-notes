# 📘 Financial Mathematics  

This repository contains my notes, demonstrations, and exercises in **financial mathematics**.  
It is an evolving project where I progressively build a course that combines theoretical insights with practical applications.  

## 📂 Content  

- **Theoretical notes**: definitions, theorems, and mathematical proofs.  
- **Practical examples**: applications of financial models with simulated or real data.  
- **R code**: implementations of tools (option pricing, discounting, Brownian motion, etc.).  
- **Exercises**: guided problems and solutions to practice the concepts.  

## 🛠️ Repository Structure  

- `financial_math.Rmd` → Main R Markdown file (regularly updated).  
- `financial_math.pdf` → PDF version generated from the Rmd for easier reading.  
- `scripts/` → Additional R scripts for demonstrations.  
- `data/` → Simulated or imported datasets for examples.  

## 🎯 Goals  

- Build a clear and rigorous course in **financial mathematics**.  
- Combine **mathematical theory** with **numerical implementation in R**.  
- Provide a progressive learning path: definitions → proofs → applications → exercises.  

## 📝 Topics (to be added gradually)  

- Simple and compound interest  
- Discounting and Net Present Value (NPV)  
- Continuous compounding and discounting  
- Annuities and perpetuities  
- Bonds and duration  
- Portfolios and expected return  
- Black-Scholes model and options  
- Stochastic processes and Brownian motion  
- Monte Carlo simulation applied to derivatives  

## 🚀 How to Use  

This repository is **work in progress**: each update of my notes will appear here as both **Rmd** and **PDF** files.  
You can:  
- Read the PDF to follow the course progression.  
- Explore the Rmd and scripts to see the code and technical details.
